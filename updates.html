Updates

1) Analogy: Car Maintenance

Cars need regular service to fix issues and run well. Devices need updates for the same reason: to fix security holes and keep things smooth.

2) Why Updates Matter

Updates patch vulnerabilities that attackers look for. Delaying updates is like leaving a known window unlocked. Turn on automatic updates for your system and your apps.

3) Do This Now

1Windows: Settings → Windows Update → Turn on “Get the latest updates” → Check for updates.

2macOS: System Settings → General → Software Update → Automatic Updates On.

3Apps: In Microsoft Store or App Store, enable automatic app updates. For browsers, keep auto‑update on.
