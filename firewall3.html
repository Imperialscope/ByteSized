Firewalls — Level 3

1) Open Windows Security

  • 1 Press Start and type Windows Security. Open it. This is where Windows Defender Firewall lives.
  • 2 Choose Firewall & network protection.
  • 3 Review Domain, Private, Public network status.

2) Allow an App Through Firewall

  • 1 Click Allow an app through firewall.
  • 2 Click Change settings, tick the app, and choose Private and/or Public.
  • 3 Click OK to apply.

3) Create an Inbound Port Rule

  • 1 Click Advanced settings to open Windows Defender Firewall with Advanced Security.
  • 2 Choose Inbound Rules → New Rule...
  • 3 Select Port → TCP → Specific local ports (e.g., 8080).
  • 4 Choose Allow or Block → Apply to profiles → Name the rule → Finish.

4) Create an Outbound Rule

  • 1 In the left panel choose Outbound Rules → New Rule...
  • 2 Choose Program or Port, set the path/port and protocol.
  • 3 Pick Block to prevent traffic or Allow to permit, then scope to profiles.
